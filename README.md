# Gym Website

A responsive and modern gym website built using HTML5 and CSS3. The website provides information about gym services, trainers, pricing plans, and contact details through a clean and user-friendly interface.

## Features

* Responsive design
* Modern and attractive UI
* Home page with gym introduction
* About Us section
* Services section
* Trainer profiles
* Pricing plans
* Contact page
* Easy navigation between pages

## Technologies Used

* HTML5
* CSS3

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/gym-website.git
```

2. Navigate to the project folder

```bash
cd gym-website
```

3. Open `index.html` in your browser.
